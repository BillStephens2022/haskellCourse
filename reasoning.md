# Equational Reasoning

## Properties of algebraic functions

1. **Commutativity**:
   - For addition: x + y = y + x
   - For multiplication: x * y = y * x

2. **Associativity**:
   - For addition: x + (y + z) = (x + y) + z
   - For multiplication: x * (y * z) = (x * y) * z

3. **Distributivity**:
   - Distributive property of multiplication over addition: 
        x * (y + z) = (x * y) + (x * z)
   - Distributive property of addition over multiplication:
        (x + y) * z = (x * z) + (y * z)

Example:
(x + a) (x + b)
= (x + a) x + (x + a) b
= x^2 + ax + xb + ab
= x^2 + ax + bx + ab
= x^2 + (a + b)x + ab
